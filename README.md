### Block cursor settings

````
"block-cursor":
  global:
    blinkOn:
      backgroundColor:
        red: 0
        green: 0
        blue: 0
        alpha: 0
      borderColor:
        red: 255
        green: 255
        blue: 255
        alpha: 0.7
      borderStyle: "i-beam"
      borderWidth: 3
```
